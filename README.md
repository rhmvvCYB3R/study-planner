# Project Topic
**Study Planner Mobile App**

## Short Description
A mobile app to help students organize their study schedule, track tasks and goals, and receive reminders for important deadlines. The main idea is to combine planning, progress tracking, and motivation in one app.

## Key Features
- **Task Management:** Create and edit study tasks and schedules  
- **Categorization:** Organize tasks by subject or activity type  
- **Reminders:** Notifications for upcoming deadlines  
- **Progress Tracking:** Mark tasks as complete and monitor progress  
- **Offline Access:** Local data storage for working without internet  
- **Responsive Design:** Works on different screen sizes and orientations
